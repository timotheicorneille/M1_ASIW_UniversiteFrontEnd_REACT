# cours

To install dependencies:

```bash
bun install
```

To run frontend:

```bash
cd packages/front
bun run dev
```

To run backend:

```bash
cd packages/back
bun run dev
```

This project was created using `bun init` in bun v1.2.23. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
